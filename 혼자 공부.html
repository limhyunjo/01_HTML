기본 용어 
        [속성]
        (속성: 시작 태그에 작성하는 내용)]
        
        1. src : 삽입할 이미지의 경로를 작성하는 속성
                (파일 위치, 인터넷 주소)
                
        2. width / height : 너비 / 높이를 지정하는 속성
        
        3. alt (대안) : 이미지 설명을 작성하는 속성
             -> 이미지가 출력되지 않은 경우 화면에 출력됨
             + 웹 접근성(스크린 리더가 이미지 설명을 읽어줌)
